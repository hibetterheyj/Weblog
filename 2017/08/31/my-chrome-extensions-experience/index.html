无扩展，不 Chrome

没有插件的chorme是低效且不完整的！
除了快速流畅的浏览体验，各种强大的扩展也是许多人选择 Chrome 的重要原因。

      到了大学，电脑愈发普及，人们借助浏览器来上网的时间也愈发的增加，一大群极客们借助着Chrome的拓展程序版块使得浏览器的使用体验越发顺畅，工作也得以高效的完成，因此我就秉持着“把工具用好就好”的心态，选出自己喜爱的Chrome插件以飨读者！

PS: 以下下载链接均为chrome官方，如若无法下载，访问Chrome 扩展程序下载可以解决相关问题，下载到正常安全的插件。

插件介绍

浏览器功能强化

  • Steward

      说到启动器（LaunchPad），Windows上较为有名当属Wox。它是由一个命令输入框，以及一个查询结果下拉列表组成，只需要一个命令就能让浏览器去完成一系列操作，如同你的管家一样，自然是很多人心目中的神器。而 Steward 便是 Chrome 浏览器里的类 Alfred 启动器。

在 Chrome 中，我们通过 Steward 简单地输入某些命令，就能完成扩展、书签、标签页管理等大部分浏览器操作。它之余效率不高还很手贱的而言，便意味着可以一键屏蔽许多网站，火箭般提高使用浏览器的效率。其中的wfs工作流命令是我目前的最爱！

插件分为原始具有三种模式的 Steward 版本以及省去New Tab功能的 Browser Alfred 版本。

      相关文章可阅读：

  • Steward - Gitbook
  • 在 Chrome 里用「Alfred」是什么体验？Steward 把效率启动器带进了浏览器 - 少数派
  • Chrome 命令启动器 Steward 攻略 – 三种模式各有神通 - 少数派
  • Chrome 命令启动器 Steward 的进阶用法：Workflow 批量操作 - 少数派

下载链接：Chrome Web Store – Steward

Chrome Web Store – Browser Alfred

 

  • 一键管理所有扩展

      当越来越多的 Chrome Extensions 被安装在Chrome上，浏览器的功能不得不被拖慢。这款插件——一键管理所有扩展支持包括一键禁用所有扩展、一键开启原有扩展、扩展的逐个禁用和开启、右击扩展等功能。此插件为以前的一键禁用所有扩展-加强版的最新版本。

下载链接：Chrome Web Store – 一键管理所有扩展

 

  • OneTab

      OneTab可以节省内存，并减轻标签页混乱现象。其他人推荐：Chrome这个个吃内存的怪兽啊，有了这个神器，网页打开再多都不怕啦！必装推荐！

下载链接：Chrome Web Store – onetab

 

  • Infinity新标签页

      Infinity实现了一个更好用，更强大的新标签页：自由定制chrome新标签页;开启页面添加时代，基于html5的Chrome扩展程序。相比Chrome自带的新标签页，可以通过Infinity自定义添加喜爱的网站，并且集成了天气，待办事项，笔记，插件扩展及历史记录管理、个性化搜索等功能。

下载链接：Chrome Web Store – infinity-new-tab

 

  • LastPass: Free Password Manager

      LastPassChrome上最好用的密码管理软件，支持密码自动填充、生成密码。并且可以通过注册账户，多平台保存密码,安全访问计算机以及移动设备。

下载链接：Chrome Web Store – lastpass-free-password-ma

 

  • DomainFlag

      DomainFlag可显示当前网站服务器的国家和 MozRank 或 Alexa 排名等 SEO（Search Engine Optimization，搜索引擎优化） 信息，并在地址栏显示当前网站服务器的所在国家和网站排名（MozRank / Alexa 排名）。还可以实现快速访问关联页面、生成页面二维码、自定义链接模板等紧俏功能。总体而言，这是一个能够满足我好奇心的插件！

      相关文章可阅读：DomainFlag - 轻量、好用的 Chrome SEO 扩展

下载链接：Chrome Web Store – domainflag

 

 

搜索相关功能增强

  • Lookup Companion for Wikipedia

      我认为它是维基百科用户必备扩展。功能全面：可以保存您最近的维基查询记录，多语种，后退/前进等等……最棒的则是：可以选择文本然后用右键点击搜索！

下载链接：Chrome Web Store – lookup-companion-for-wiki

 

 

网页效果优化

  • Stylish


Baidu lite效果图

      许多网页的视觉风格是用 CSS 文件进行管理的，替换网页本身的 CSS 可以达到美化的目的。通过安装 CSS，让百度清爽无比（裂墙推荐！），让github变得更宽更美，让 Google 用上动漫特效背景…… Stylish 作为一个 CSS 管理器，可以让替换 CSS 变得简单。

      相关文章可阅读：少数派：不喜欢某个网站的样子？用 Stylish 给它一键「换肤」

下载链接：Chrome Web Store – stylish-custom-themes-for

 

  • Wikiwand


Wikiwand效果图

      多年来，维基百科网页的视觉风格坚定不移地停留在 Web2.0 时代（强烈吐槽！！！）。Wikiwand 实现了视觉上的提升，并添加了一些人性化的交互：比如超链接悬浮窗，类似 Kindle 的书签功能，搜索联想，黑色模式等等…

      相关文章可阅读：少数派：轻、快、美，浏览维基百科的魔杖：Wikiwand

下载链接：Chrome Web Store – wikiwand-wikipedia-modern

 

  • 眼不见心不烦（新浪微博）


【眼不见心不烦】效果图

      对于越来越臃肿的微博，我们还是眼不见心不烦吧。「眼不见心不烦」由国人开发，可以高度自定义微博的显示样式。安装成功后在页面右上角的「齿轮」便可打开设置页面。新浪微博非官方功能增强插件，可以无限制地屏蔽关键词、用户、来源，去除页面广告和推广微博，反刷屏，还您一个清爽干净的微博！

下载链接：Chrome Web Store – 眼不见心不烦（新浪微博）
