      作为技术小白，遇到电脑上的很多问题不得不度娘，跳坑经历常有，不妨一一总结，以备他人不时之需。

免费科学上网：lantern

      人民不断增长的科学上网需求与日益强大的GFW是现阶段想要“睁眼看世界”的主要矛盾，蓬勃发展的vpn产业由此催生。一大堆收费的、免费的软件进入了人们的视野，不过“好用的vpn总是相似的，但不好的却是各有各的难用”，所以当现在还有人问我用什么，我还是一如既往的推荐Lantern。

安装使用过程


图1.1 lantern官网首页

      由于其官网受到强大的GFW阻隔，不过程序员还是异常强大的，在github上开放了一个专门的repository——getlantern/forum（手机端建议复制网址访问，微信端被举报屏蔽！），以供需要下载。

      各种系统设备（包括安卓，Windows，macOS甚至Linux）应该都能找到较为合适的版本。每个月有一定的免费高速流量，全部使用完后可以使用“设置-高级设置-代理全部流量”的操作获得不错的访问速度（尤其是访问如Coursera这样的学习视频网站）。

      在Windows端使用过程中可能会带来“国内网站无法正常访问或访问较慢”的问题。我当时也是因为这个着急了很久，度娘来的方法十分见效（详见相关网址2）。

      真正厉害的大神当然推荐其他更高级省心的方法，比如我就十分心水车队高效快速且自带vpn的局域网⁽⁽(ཀ д ཀ)⁾⁾

国外慕课下载神器：Coursera-dl

      现在越来越多的人使用国外的慕课网站尤其由大名鼎鼎的Andrew Ng创办的Coursera来自学各类课程。但是毕竟我们在墙内总会遇上问题，而这其中最主要的就是网站、视频访问速度太慢，随之而来的解决方法就是利用小技巧将讲义、视频还有字幕文件下载下来。此方法还能方便本地随时查看，减小因网络限制带来的影响！

安装使用过程

      经过多次跳坑试坑，最终确定
知乎：如何批量下载 Coursera 视频、字幕、教材？– Li Ming 的回答 方为正确的操作，而目前较为高票的回，偏听则暗啊！）

安装

      前置条件为Python的安装，建议为较新的Python3.X版本。

      成功安装Python后，以管理员权限打开命令行cmd输入pip install coursera-dl，慢慢等待下载的完成。

操作

      1.（是否管理员权限皆可）打开命令行cmd输入以下代码,下载地址为命令行打开地址，命令行内容可以简单查看我的命令行入门笔记。

coursera-dl -u 用户名 -p 登陆密码 课程名称

      2.什么是课程名称？      点开想学习的课程(我一般注册课程后下载)，查看它的网址，比如最出名的Andrew Ng的机器学习课程网址

http://www.coursera.org/learn/machine-learning 
%本课程名称即为learn以后的一段内容：machine-learning

      3.所以输入命令行即为

coursera-dl -u your_username -p your_password machine-learning

      4.最后也是最重要的一步——由于种种原因，下载未能完全（比如我就遇到只下载了讲义，无法成功下载视频的情形），那么则需要更新电脑的host文件，避免GFW的DNS污染（天朝真是威武！），网址详见相关网址。

      5.进阶操作

coursera-dl -u <user> -p <pass> 课程名称
% 一般操作          

coursera-dl -u <user> -p <pass> 课程1 课程2 ... 课程n
% Multiple classes多课程一起下

coursera-dl -u <user> -p <pass> -f "ppt" qcomp-2012-001
% Download only ppt files: 只下某一类型如ppt    

coursera-dl -n -b ni-001
% Get the preview classes:下载课程预览?     

coursera-dl -n --path=C:\Coursera\Classes\ comnetworks-002
% Specify download path: 下载位置      

coursera-dl --help
%Display help:  显示帮助

      更多选项请参考项目github页面

      似乎网上还有另一个外国慕课平台edX的下载工具，我以后会去一探究竟的，此处为相关github项目页面。
